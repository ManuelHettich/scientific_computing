Implement the gradient descent solver as presented in the lecture.

Besides the function parameters x and b, you will have to store only the residual r, and the product Ar = A * r.

The stopping criterion should be the relative residual error, i.e., norm(A * x - b) / norm(b) < _error. Note that r = b - A * x is computed anyway, as well as r * r, so you don't have to compute (A * x - b) again for the error check.
